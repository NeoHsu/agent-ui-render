# Compact v2 Charts

Compact input version 2 adds explicit chart families while preserving the
project's small, governed, dataset-oriented authoring boundary. The Vue
renderer uses Vega-Lite internally, but Vega-Lite JSON is never accepted as
model-authored input.

## Flow

```text
+-------------------------+
| Compact v2 tuples       |
| shared datasets + refs  |
+------------+------------+
             |
             v
+-------------------------+
| Rust validation         |
| indexes, types, options |
+------------+------------+
             |
             v
+-------------------------+
| Trusted Vega-Lite spec  |
| generated by runtime    |
+------------+------------+
             |
      +------+------+
      |             |
      v             v
+-----------+ +-------------------+
| rich HTML | | static HTML      |
| SVG chart | | native or table  |
+-----------+ +-------------------+
```

## When to use v2

Keep using version 1 for ordinary trend, comparison, distribution,
composition, relationship, and records views. Use version 2 when an explicit
chart such as histogram, area, heatmap, boxplot, error band, bubble, Gantt,
candlestick, layer, or facet is required.

```json
{
  "version": 2,
  "d": [
    [
      "latency",
      [["service", "s"], ["latency_ms", "n"]],
      [["api", 182], ["api", 205], ["worker", 311]]
    ]
  ],
  "v": [["hist", 0, 1, {"bn": 20}], ["box", 0, 0, 1]]
}
```

## Coverage

- Line, step, slope, bump, trail, regression, and LOESS
- Area, stacked/normalized area, streamgraph, and horizon templates
- Grouped/stacked/normalized/diverging bar
- Histogram, density, dot, strip, boxplot, QQ
- Scatter, bubble, parallel coordinates, ternary
- Heatmap, 2D histogram, and mosaic
- Pie, donut, and radial charts
- Error bar, error band, and confidence interval
- Gantt, waterfall, bullet, ranged dot, and candlestick
- Text, tick, and rule plots
- Layer, facet, repeat, concat, and controlled interactions

Explicitly unsupported:

- Images and image-based scatter
- Isotype/pictogram charts
- Geoshape and every map/geographic projection
- Raw Vega-Lite specifications, expressions, URLs, hrefs, or arbitrary signals

`skills/agent-ui-render/references/charts-v2.md`.

## Rendering behavior

`render html` embeds the bundled Vega/Vega-Lite runtime and renders trusted
specifications as SVG without external network access. `render static-html`
continues to use native Rust charts where available and otherwise emits a
visible explanation plus an accessible projected table so source facts remain
available without JavaScript.

## Rich interaction behavior

Rich HTML keeps SVG output interactive. Data marks expose a structured,
series-aware tooltip and pointer feedback. Set `sel` when the report needs a
persistent interaction:

- `hover`: highlight the nearest mark while the pointer is over the plot.
- `click`: select a mark, dim the remaining marks, and clear on double-click.
- `brush`: drag an interval to select marks inside it.
- `zoom`: drag an interval or use minus/plus controls to change bound
  scales.
- `legend`: select a generated series through its legend and dim the others.

A consistent toolbar lane keeps configured charts aligned without covering
legends. Persistent selections expose **Clear** and can also be reset with
Escape while the chart has focus. Vega-generated SVG accessibility labels
remain available to assistive technology. `render static-html` intentionally
has no interaction and preserves
facts through its native chart or accessible table fallback.

## Compatibility

Version 1 remains stable and unchanged. Version 2 accepts the v1 semantic view
tuples in addition to explicit chart tuples, allowing reports to mix concise
automatic views with advanced chart requests.
