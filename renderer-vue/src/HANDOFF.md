# Agent UI Renderer handoff

Keep this directory beside the generated wrapper component. Install compatible
peer dependencies before building:

```bash
bun add vue@^3.5.17 vega@^6.2.0 vega-lite@^6.4.3
```

The renderer accepts validated normalized Agent UI reports. Vega-Lite
specifications are generated by the trusted `agent-ui-render` planner; do not
replace them with untrusted raw specifications.
